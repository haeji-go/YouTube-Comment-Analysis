Youtube 댓글분석 및 채널추천 서비스

[서비스 소개]

유튜브 댓글을 실시간으로 수집하여 분석해주는 서비스입니다.
유튜브 영상 url입력만으로 한 눈에 보기 힘든 유튜브 댓글의 분석된 결과를 확인해보세요!😊
로그인없이 새로운 댓글 작성만으로 새로운 채널을 추천받아보세요! 😊

해당 서비스는 아래와 같이 크게 두 부분으로 이루어져있습니다.

(1) 유튜브 댓글 분석

✔ 키워드 클러스터링
전체 댓글들을 3개의 주제(토픽)으로 분류하여 토픽의 분포와 해당 토픽별 단어의 분포를 시각화한 분석기법

✔ 감성분석
전체 댓글에 대하여 긍정, 부정, 중립을 판별하여 그 비율을 도넛차트를 이용하여 시각화

✔ 워드클라우드
전체 댓글의 키워드를 직관적으로 파악할 수 있도록 비중이 높은 단어를 글자크기와 색을 통해 강조하는 시각화

(Tip) 도넛 차트의 각 긍정, 부정, 중립을 누르면 해당 감성별 워드클라우드를 확인할 수 있습니다. 😊
      예. 긍정 도넛 조각을 클릭 🠖긍정 판단에 영향을 준 단어들로 구성된 워드클라우드 확인가능

(2) 댓글 분석 기반의 채널 추천

분석한 영상에 대해 새로운 댓글을 입력하면, 기존 댓글과 새로운 댓글을 분석하여 이웃을 선정하고,
해당 이웃의 구독정보를 수집하여 새로운 채널을 추천해드려요!
(Tip) 댓글이 적은 경우, 기존 댓글과 다른 양상의 댓글을 입력할 경우 채널 추천 성능이 제대로 안 나올 수 있어요!😥


[서비스 이용]

(1) 유튜브 댓글 분석

[step1] 유튜브 영상의 url 복사 (youtube로 이동하기)
[step2] 복사한 url을 입력 후, 엔터
[step3] 분석 결과 확인

(Tip) step2➜step3 : 댓글 수집 🠖 텍스트 전처리 🠖 텍스트 분석 🠖 분석결과시각화
- 진행 과정을 확인할 수 있어요! 😊
- 여러 과정이 진행되다 보니, 댓글이 많을수록 시간이 더 소요됩니다!😥

(2) 댓글 분석 기반의 채널 추천

[step1] 분석 페이지에서 새로운 댓글 입력 후, 엔터
[step2] 댓글 분석 기반으로 추천된 채널 확인

(Tip) step1➜step2 : 새로운 댓글과 기존 댓글간의 유사도 계산 🠖 기존 댓글 작성자들의 구독 정보 수집 🠖 추천 알고리즘
- 위의 과정들이 진행되므로 시간이 소요됩니다!😥
